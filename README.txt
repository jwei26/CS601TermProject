Personal Portfolio Website

This project is a personal portfolio website designed to showcase my skills, experiences, and projects. It serves as a comprehensive digital resume, providing a quick and easy way for potential employers or collaborators to get an overview of my professional capabilities and past work.

Technologies

The website is created with:

HTML
CSS
ES6 JavaScript
Vue.js
Fetch API
Features

The website contains four main sections:

Home: A brief introduction and welcoming section.

About Me: A detailed description of my skills, background, and interests. Skills are displayed dynamically using Vue.js.

Portfolio: A showcase of my past projects. Project data is fetched dynamically from an external JSON file using the Fetch API and displayed using Vue.js.

Resume: Detailed work history and education background. Work experience data is fetched dynamically from an external JSON file using the Fetch API and displayed using Vue.js.
